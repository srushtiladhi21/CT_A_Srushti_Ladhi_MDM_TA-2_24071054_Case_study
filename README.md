# Zepto Homepage Clone with Recommendation System

## Project Title

Zepto Homepage Clone with Recommendation System and Working Cart

## Project Description

This project is a frontend implementation of a Zepto-like homepage with a recommendation system and working cart functionality.
The main objective of this project is to understand how modern e-commerce websites work and how recommendation systems suggest products to users.

The project is inspired by the design and functionality of Zepto and includes product sections, categories, banners, cart system, and a recommendation engine based on product categories.

This project is developed for Teacher Assessment (TA) to demonstrate frontend development and recommendation system implementation using HTML, CSS, and JavaScript.

## Recommendation System

The recommendation system suggests similar products based on category.

### Working

1. User clicks product
2. System finds product category
3. Similar category products are selected
4. Recommended products are displayed

### Example

Milk → Butter, Cheese
Chips → Biscuits
Bread → Cake

---

## Algorithm Used

### Content-Based Filtering

This algorithm recommends products based on product category.

### Steps

* Store product data
* Detect selected product
* Identify category
* Match similar category
* Show recommended products

### Time Complexity

O(n)

because all products are checked.
